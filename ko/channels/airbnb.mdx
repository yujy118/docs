<Warning>
  에어비앤비 연결 전 네이버 간편 로그인이 가능한 계정을 준비해 주세요.

  1. VCMS에서 에어비앤비를 연동하려면 네이버 계정만 사용할 수 있습니다.
  2. 네이버 간편 로그인이 가능한 에어비앤비 계정을 준비해 주세요.  (\*네이버 단체 계정은 간편 로그인을 지원하지 않아 사용할 수 없습니다.)
</Warning>

### 에어비앤비 로그인을 위한 `네이버 간편 로그인` 설정

<AccordionGroup>
  <Accordion title="1. 이에일 주소를 네이버 간편 로그인으로 변경하기">
    <Check>
      이메일 주소를 네이버 간편 로그인으로 변경하기

      1. 메뉴 \> 계정 설정 \> [**개인 정보**](https://www.airbnb.co.kr/help/article/362)를 클릭하세요.
      2. 이메일 주소로 가서 **수정**을 클릭하세요.
      3. 이메일 주소를 변경한 후 **저장**을 클릭하세요.
    </Check>
    <Info>
      에어비앤비 도움말 바로가기

      - [이메일 주소 수정](https://www.airbnb.co.kr/help/article/362)
    </Info>
  </Accordion>
  <Accordion title="2. 네이버 간편 로그인 계정을 공동 호스트로 초대하기">
    <Check>
      네이버로 생성한 에어비앤비 계정을 사용중인 숙소의 공동 호스트로 초대해 주세요.

      1. 상단 메뉴에서 [숙소]를 클릭한 뒤, 권한을 넘기려는 숙소를 선택하세요.
      3. [숙소 페이지 에디터]로 이동해 왼쪽 메뉴에서 [숙소]를 클릭하세요.
      4. 메뉴에서 [지인 초대]를 선택하세요.
      5. 새로 만든 네이버 계정의 국가/지역 + 전화번호 또는 이메일을 입력하고 [다음]을 클릭하세요.
      6. 새 계정에 부여할 접근 권한 범위를 선택한 뒤 [다음]을 클릭하세요.
      7. 초대 내용을 확인한 후 [보내기]를 클릭해주세요.
      8. 에어비앤비 승인이 완료되면, 해당 계정으로 VCMS에서 네이버 계정으로 연동해 주세요.
    </Check>
    <Info>
      에어비앤비 도움말 바로가기

      - [숙소 리스팅에 공동 호스트 추가하기](https://www.airbnb.co.kr/help/article/1244)
    </Info>
  </Accordion>
  <Accordion title="3. 네이버 외부 애플리케이션 암호 설정하기">
    <Check>
      네이버 연동을 위한 외부 애플리케이션 비밀번호 생성을 진행해 주세요.
    </Check>
    <Steps>
      <Step title="1. 네이버 외부 애플리케이션 비밀번호 만들기">
        <Check>
          네이버 외부 애플리케이션 비밀번호란, 외부 앱에 네이버 계정으로 로그인할 때 사용하는 **외부 앱 전용 비밀번호**를 말합니다.

          - 네이버 연동 시 보안을 위해 2차 인증과 더불어 외부 애플리케이션 비밀번호를 생성 후 사용해 주세요. 단, 단체 회원의 경우, 해당 설정이 불가능하니 개인 로그인 계정을 준비해 주세요.
          - 👉 [네이버 고객센터 도움말 바로가기](https://help.naver.com/service/5640/contents/8584?lang=ko&osType=COMMONOS)
        </Check>
      </Step>
      <Step title="생성 방법">
        1. [네이버](https://nid.naver.com/user2/help/myInfo.nhn?m=viewSecurity&lang=ko_KR) 에 접속합니다.
        2. **기본보안설정** 항목에서 **2단계 인증** 메뉴에 위치한 **관리** 버튼을 클릭합니다.
        3. 비밀번호를 입력하고, **확인** 버튼을 클릭합니다.
        4. 아래의 그림과 같이 종류를 선택하고, 비밀번호를 생성하시면, 새 애플리케이션 비밀번호가 생성됩니다.

        ![image (60).png](/images/image\(60\).png)

        5. 복사한 외부 애플리케이션 비밀번호를 네이버 ID 비밀번호로 사용해 주세요.

        <Warning>
          외부 애플리케이션 비밀번호를 잊어버린 경우, 다시 확인할 수 없으며 새롭게 생성해야 합니다.

          - 만약, 2차인증 설정을 해제한 경우에도 외부 애플리케이션 비밀번호가 삭제되니 주의해 주세요.
        </Warning>
      </Step>
    </Steps>
  </Accordion>
</AccordionGroup>
