<Check>
  네이버 연동을 위한 외부 애플리케이션 비밀번호 생성을 진행해 주세요.
</Check>

<Info>
  사용중인 네이버 계정이 아래 사진과 같이 `단체 회원`인 경우, 애플리케이션 암호 설정 없이 채널을 연결해 주세요!

  ![Image](/images/image-19.png)
</Info>

<Steps>
  <Step title="1. 네이버 외부 애플리케이션 비밀번호 만들기">
    <Check>
      네이버 외부 애플리케이션 비밀번호란, 외부 앱에 네이버 계정으로 로그인할 때 사용하는 **외부 앱 전용 비밀번호**를 말합니다.

      - 네이버 연동 시 보안을 위해 2차 인증과 더불어 외부 애플리케이션 비밀번호를 생성 후 사용해 주세요. 단, 단체 회원의 경우, 해당 설정이 불가능하니 개인 로그인 계정을 준비해 주세요.
      - 👉 [네이버 고객센터 도움말 바로가기](https://help.naver.com/service/5640/contents/8584?lang=ko&osType=COMMONOS)
    </Check>
  </Step>
  <Step title="생성 방법">
    1. [네이버](https://nid.naver.com/user2/help/myInfo.nhn?m=viewSecurity&lang=ko_KR) 에 접속합니다.
    2. **기본보안설정** 항목에서 **2단계 인증** 메뉴에 위치한 **관리** 버튼을 클릭합니다.
    3. 비밀번호를 입력하고, **확인** 버튼을 클릭합니다.
    4. 아래의 그림과 같이 종류를 선택하고, 비밀번호를 생성하시면, 새 애플리케이션 비밀번호가 생성됩니다.

    ![image (60).png](/images/image\(60\).png)

    5. 복사한 외부 애플리케이션 비밀번호를 네이버 ID 비밀번호로 사용해 주세요.

    <Warning>
      외부 애플리케이션 비밀번호를 잊어버린 경우, 다시 확인할 수 없으며 새롭게 생성해야 합니다.

      - 만약, 2차인증 설정을 해제한 경우에도 외부 애플리케이션 비밀번호가 삭제되니 주의해 주세요.
    </Warning>
  </Step>
</Steps>

### 재고/요금 동기화 시 주의 사항

<Check>
  네이버 객실 목록에서 오늘의 딜(00:00 미판매시 설정된 할인율 적용), 공휴일 일정을 등록하셨다면 주의해 주세요!

  - 재고/요금 동기화 시 오늘의 딜, 공휴일 일정이 삭제되고 VCMS 판매관리 화면의 재고/요금으로 변경됩니다.

    />
</Check>

<Warning>

</Warning>
