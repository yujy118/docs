<Check>
  현재 모바일 전용 앱(App)은 지원되지 않습니다.

  - 모바일 기기에서 https://vcms.io 로 접속하시면 재고/요금 조절이 가능합니다.
  - 단, 재고/요금 설정 시 제한된 기능으로 사용할 수 있기 때문에 판매 관리 화면의 재고/요금 조절을 제외한 기능은 PC 환경에서 사용해 주세요.
</Check>

## PC & 모바일 환경 비교

### 판매 관리

|                                                               | `💻PC 환경` | `📱모바일 환경` |

| 판매 상태                                                         | O         | O          |
| 잔여                                                            | O         | O          |
| 입/퇴실 시간                                                       | O         | X          |
| 요금                                                            | O         | O          |
| 일괄 변경                                                         | O         | X          |
| <Tooltip tip="화면에 표시할 항목을 자유롭게 선택할 수 있어요.">화면 보기 설정</Tooltip> | O         | X          |
| 필터                                                            | O         | X          |

<Note>
  모바일 환경에서는 판매 상태, 잔여, 요금 변경을 지원합니다.
</Note>
